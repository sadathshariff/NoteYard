<div align="center">
NoteYard a place where you can dump all your Notes and Tasks
</div><br/>

## **How to install and run locally ?**

```
$ git clone https://github.com/sadathshariff/NoteYard.git
$ cd NoteYard
$ npm install
$ npm start
```

### Live Link

## [NoteYard](https://note-yard.vercel.app/)

## Features

- Home Page
- Archive Page
- Trash Page
- Authentication
  - User Signup
  - User Login
- Filter
 - By Label
 - By Priority
 - By Date

## Built With

- React JS
- React Router
